## 🚀 Features

- ✅ JWT Authentication (Login/Register)
- 💬 Real-time chat with dummy bot
- 🌓 Light/Dark mode toggle
- 📄 Export chat to text file
- ⚡ Prompt templates for quick starts
- 🧼 ESLint + Prettier configured
- 🔐 Protected routing

- React + TypeScript

- React Router
- Axios
